# 2×3轮式/半履带式车族 代号：山猫



采用了**2×3轮式/半履带式多用途底盘**与**模块化设计**的山猫能满足在战术上大部分需求；山猫非常擅长在复杂环境下作战也能作为迂回单位进行使用，作为轻型AFV的山猫缺陷也很明显，它不适合与其他大型AFV进行正面作战，当然作为一名指挥官只要没犯傻，都知道这一点。

**底盘 长:宽:高=12：16：3.8**

**整车设计极限 长:宽:高=12：22：8**

**底盘质量：12t**

**最高载重量：48t**
