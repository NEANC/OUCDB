# OUCDB单位设定集-旧世代-地面单位
**旧世代**的单位由于各式各样的原因还在服役，即使目前采用了无人机作为主力战争兵器也无法取消对这些单位的需求，所以我们的工程师对这些单位进行升级优化使这些单位能够适应并且匹配我们的战争模式。


## 军用陆地单位结构图
> 陆军单位(族)
>> 2×3轮式/半履带式
>>> 4×4轮式/半履带式/履带式
>>>>> 重型履带式
>>>>>> 陆地巡洋舰
>>>>>>> 陆地战列舰/陆地航空母舰
>>>>>>>> 机动工厂/运载舰
