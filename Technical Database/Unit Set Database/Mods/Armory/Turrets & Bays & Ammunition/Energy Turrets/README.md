# 能量武器 / Energy Turrets

激光与离子，粒子，声波等**定向能量武器**的集合，包括定向 EMP 武器。

## 能量武器命名规则

**XXXX-00**  
1X 使用范围 O 轨道内 P 星球内 D/I 宇宙  
2X 战术 T/战略 S  
3X 武器类型 E 能量 E-EMP  
4X 具体分类 L 激光 I 离子 P 粒子 S 声波 [特殊 EMP 混合炮无该分类]  
00 能级
