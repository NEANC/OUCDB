# 能量武器 / Energy Turrets

激光与离子，粒子，声波等**定向能量武器**的集合，包括定向EMP武器。

## 能量武器命名规则

**XXXX-00**  
1X 使用范围 O轨道内 P星球内 D/I宇宙  
2X 战术T/战略S  
3X 武器类型 E能量 E-EMP  
4X 具体分类 L激光 I离子 P粒子 S声波  [特殊 EMP 混合炮无该分类]  
00 能级
