# 技术资料库 - 单位设定集 - 模块标准化 - 武器库 - 火炮 / 发射器 与配套弹药

本库是**武器库**的底层定义集之一

## 火炮命名规则

XXXX-000Y

- 1X
  - 使用范围
    - O 轨道内
    - P 星球内
    - D/I 宇宙

- 2X
  - 战术 T
  - 战略 S

- 3X
  - 武器类型
    - E 能量
    - H 混合
    - P 射弹
    - E-EMP

- 4X
  - 具体分类
    - L 激光
    - I 离子
    - P 粒子
    - S 声波
    - C 加农炮
    - H 榴弹炮
    - E 磁轨炮
    - 特殊
      - EMP 与混合炮无该分类

- 000
  - 口径

- Y 炮管数
