# 胖墩 DMF-45 型 机动工厂

![DMF-45型 机动工厂 侧视图](./img/DMF-45.jpg)

**DMF-45 型 机动工厂**与**DVC-33 型 运载舰**是两个交叉职能的超重型陆地机动平台，**DMF-45 型 机动工厂**主要负责后勤与战略支持方面；它能在资源充足的情况下快速生产**超重型平台以下的各级陆地 / 空中单位**。

**工业系统第二序列**  

「灰蛊」与其说是纳米机器人集群还不如说是活着的金属，与传统记忆金属不同「灰蛊」具有生物生命体才具备的特征，并且还兼具了无机物的特性，因此我们的科学家对此进行大量的研究，但成就不大，不给我们已经掌握了如何制备与使用「灰蛊」的能力了。

我们的科学家使用由「灰蛊」制成的纳米机器人集群，能兼具纳米机器人的特性时也具备了「灰蛊」的特性
