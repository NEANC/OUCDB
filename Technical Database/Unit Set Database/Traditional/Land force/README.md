# OUCDB单位设定集-传统型-地面单位

# 设计指标

用于填补无人机无法满足的战术需求（例如前线指挥，解救人质，巷战等特殊环境）基于这些特殊的需求，我们的工程师采用了严苛的**AA04标准（四倍能量冗余，四倍物质冗余，四倍容纳冗余，等四倍冗余标准）**。由于无法确定作战时的特殊环境与战术需求，完全按照AA04标准设计的蓝图生产会导致严重的浪费，所以前线指挥官能自行调整设计蓝图并生产。  
**注意：涉及于人员的行动必须采用AA04标准。**
