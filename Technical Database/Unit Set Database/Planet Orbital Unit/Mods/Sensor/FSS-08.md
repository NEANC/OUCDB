# FSS-08型 复合型 传感/火控系统

FSS-08 是以 FSS-04 的底盘和 FSS-06 的系统构架为基础设计的新一代通用复合传感器，用于替代超期服役的 FSS-04 系列和广受好评的 FSS-06 系列。
